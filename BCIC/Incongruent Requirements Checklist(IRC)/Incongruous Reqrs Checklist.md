# Incongruous Requirements Checklist (IRC)

| Imperative | Continuance | Verifiability | IRC Category |
|---|---|---|---|
| Shall | Below | Testable | Inconsistent with another requirement |
| Must / Must Not | As follows | Non-Testable | Unseemingly prioritized |
| Required to | Following | Testable with Clarification Needed | Ambiguous |
| Are applicable | Listed | Issues with Verifiability | Suspected incompleteness |
| Responsible for | In particular |  | Not feasible / Unrealistic |
| Will | Supported by |  | Unreadable due to typo(s) or missing word(s) |
| Should |  |  |  |
